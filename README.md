# MiraiGo-module-fortune

ID: `com.aimerneige.fortune`

Module for [MiraiGo-Template](https://github.com/Logiase/MiraiGo-Template)

## 功能

- 接收到 “求签 <求签内容>” 时，进行一次求签，并返回求签结果。

## 使用方法

在适当位置引用本包

```go
package example

imports (
    // ...

    _ "github.com/yukichan-bot-module/MiraiGo-module-fortune"

    // ...
)

// ...
```
